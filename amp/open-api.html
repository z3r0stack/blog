Open API ที่น่าสนใจ

by z3r0stack,
published on 26 Jul 2017

รายการ Open API 

Serverless Framework

API สำหรับการสร้าง Application ที่ต้องการความเร็วในการพัฒนา แบบไม่ต้องตั้งค่า server อะไรมาก คิดเงินตามการใช้งานใช้มากจ่ายมาก ใช้น้อยจ่ายน้อย ทำงานบน AWS Lambda


OpenTable

API ข้อมูลเกี่ยวกับร้านอาหารในเมืองและประเทศต่างๆ เช่น สหรัฐอเมริกา, เยอรมัน, ญี่ปุ่น, อังกฤษ, ออสเตเรีย และอื่นๆ


SF OpenData

API ข้อมูลของเมือง ซานฟรานซิสโก ซึ่งมีหลากหลายหมวดหมู่ เช่น เศรษฐกิจ, การเมือง, การขนส่ง, ความปลอดภัย, การพยาบาล, สภาพพื้นที่ และอื่นๆอีกหลายหมวด

Spotify API

API ของบริการฟังเพลงออนไลน์ ที่มีความนิยมสูงในต่างประเทศมีข้อมูลทั้งเพลง และศิลปิน หลากหลายแบบ

Imgflip API

API สำหรับการทำรูปภาพ Meme ทำ GIF จากภาพหรือวิดีโอ ทำ Pie Chart โดยสามารถใช้งานได้แบบง่ายๆ

Yelp API

API Yelp เว็บไซต์ผู้ให้บริการด้านการข้อมูล รีวิวสถานที่ท่องเที่ยวและบริการต่างๆในโซนยุโรปและอเมริกา

Twitter API

API ของ Social Network  ชื่อดัง Twitter ที่มีผู้ใช้จำนวนมากและก็มีข้อมูลให้ เล่นได้หลายรูปแบบ


Instagram API

API ของ Social Network ชื่อดังอีกตัว Instargram เป็น Social Network ที่มีภาพหลากหลายสไตล์


EventBrite API

API ของเว็บที่ให้บริการ ด้านการจัดกิจกรรม การจำหน่ายตั๋วงานอีเว้นต่างๆ


Youtube API

API ของ Youtube ที่สามารถเล่นกับข้อมูล รายการวิดีโอทั้งหลายแหล่


Rapid API

เว็บที่สำหรับค้นหา รายการ API ที่เปิดให้บริการโดยรวบรวมไว้ เยอะแยะมากมาย เรียกได้ว่าเพียบ
